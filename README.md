# Smart Library Mobile Application

## Overview
Smart Library Mobile Application is an Android-based library management system developed to simplify library operations and improve user experience through personalized book recommendations.
The application allows students to search and request books, while admins and staff can efficiently manage users, books, and return details through separate modules.
## Features
### Student Module
- User Registration and Login
- Search Books
- Personalized Book Recommendations
- View Status
- Return Details

### Staff Module
- Add Books
- Manage Book Requests
- Update Status
- Return Management
- View Student Details

### Admin Module
- Add Staff
- View Staff
- View Student Information

## Technologies Used
- Front End: XML
- Back End: Android
- Database: MySQL
- IDE: Android Studio
- Platform: Windows 10 or Higher

## System Architecture
The application follows a role-based architecture with separate modules for Admin, Staff, and Students connected through a centralized database system.

## Purpose
The main objective of this project is to automate library management processes and provide smart book recommendations to users for an enhanced reading experience.

## Future Enhancements
- AI-based recommendation engine
- Cloud database integration
- Online book reservation
- Push notifications for due dates
